## 🧠 Модель данных Ганта (важно — Алёна явно сформулировала)

В БД у каждого заказа/упаковки хранится:
- **Стартовая дата цепочки** (для заказа: `decisionDate`; для упаковки: `decisionDate`)
- **Длительности фаз** в днях (через end-даты последовательных фаз)

Фазы всегда **встык**: end одной = start следующей (одно поле в БД).

Пример для заказа:
```
decisionDate           ← start Разработки
handedToFactoryDate    ← end Разработки = start Производства
readyAtFactoryDate     ← end Производства = start ОТК
qcDate                 ← end ОТК = start Доставки
arrivalPlannedDate     ← end Доставки
```

## 🎯 Контракт жестов в Ганте/таймлайнах (главный закон)

**Каждая стрелочка соответствует ОДНОМУ значению в БД:**

| Жест | Что меняется в БД | Эффект на UI |
|---|---|---|
| **◀ самой первой плашки (Разработка)** | startField (decisionDate) | Все фазы сдвигаются на ту же дельту. Длительности всех сохраняются. = «Сдвинули цепочку во времени». |
| **▶ любой плашки (включая первую)** | endField этой фазы | Длительность ЭТОЙ фазы меняется. Соседи СПРАВА едут на дельту, их длительности сохраняются. |
| **◀ не первой плашки** | endField предыдущей фазы (= то же поле что её start) | Эквивалент ▶ предыдущей: меняется длительность ПРЕДЫДУЩЕЙ. Текущая и далее едут с длительностями. |

**Никаких clamp'ов** — даты можно тянуть и в прошлое, и в будущее сколько угодно. Если фаза перевернётся (end < start) — Алёна сама увидит и поправит.

**Drag-середины-как-move отключён.** Только за края (◀ ▶). Это явное требование Алёны.

## ⚙️ Правила (memory + рабочие)

1. **Никаких `.default()`** в Zod-схемах с `.partial()` для PATCH — обнуляют поля.
2. **Не пересоздавать БД** без явного консента. Алёна выдала: «да там все данные фейковые забиты, мы не боимся их потерять» — этот консент действует.
3. **Простой язык** для Алёны: «форма / справочник / карточка / ссылка», не «API / JWT / миграция».
4. **Короткие отчёты** после правок: «Готово. Обнови страницу. Теперь работает X.»
5. **Не переспрашивать про работу и деплой** — задачу взяла → делаю → коммит → push до прода без «можно?». PR создаётся, мержится через `gh pr merge --squash --admin`. Vercel сам выкатит.
6. **Никакой `git push --force` на main**.
7. **Drag за края, не за середину** — главный UX-закон Гант'а (см. memory `feedback_gantt_drag_edges.md`).
8. **Стиль Алёны**: скрин + короткая фраза → AI догадывается → чинит. «Делай все» = делай всё подряд без переспрашиваний.
